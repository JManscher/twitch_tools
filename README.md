# Twitch Tools

A collection of utility tools for Twitch streamers, designed to enhance stream interactivity and provide useful integrations.

## Tools

| Tool | Description |
|------|-------------|
| [MixItUpGameCommand](./MixItUpGameCommand/README.md) | A CLI tool that fetches Steam game statistics for use with Mix It Up's `/game` Twitch command. Retrieves playtime, achievements, and last played info from your Steam library. |

## Getting Started

Each tool has its own setup instructions. Click on a tool above to view its README for detailed documentation.
